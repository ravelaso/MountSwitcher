# MountSwitcher
World Of Warcraft - WOTLK Classic Addon

Switches your mounts depending on flying capabilities (fixes dalaran flying zone)

## How to

### 1 - Open the UI
```
/ms options
```

### 2 - Set your Flying/Ground mounts
Use exact name.

### 3 - Create a macro
Create a macro with the following:
```
/ms mount
```
